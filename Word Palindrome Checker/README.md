# Word Palindrome Checker
A simple C program to check if a given word is a palindrome.

## Description
The Word Palindrome Checker is a C program that checks if a given word is a palindrome. A palindrome is a word, phrase, or sequence of characters that reads the same backward as forward. For example, "level" is a palindrome.

The program compares the characters of the word from the beginning and the end, moving inward. It handles words of up to 99 characters.
